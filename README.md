# Arsenal Union

Game club web app: bron, to'lov, admin panel, Telegram bot va stansiya agent.

## Ishga tushirish

1. `server` — `npm install && npm run dev` (port **4000**)
2. `web` — `npm install && npm run dev` (port **3000**)
3. `admin` — `npm install && npm run dev` (port **3002**)
4. `server/.env` — `server/.env.example` dan nusxa oling (`CLIENT_ORIGIN` da `http://localhost:3002` bo'lishi kerak)

| Ilova | Manzil |
|-------|--------|
| Foydalanuvchi | http://localhost:3000 |
| Admin panel | http://localhost:3002 |
| API | http://localhost:4000 |

Admin kirish: telefon `+998901111111`, ism `Admin`, familiya `Union`, parol `admin1234`, secret key `arsenal-admin-secret` (`.env` dagi `ADMIN_*` qiymatlar).

Mahalliy qo'llanmalar: `docs/` (GitHubga kirmaydi).
